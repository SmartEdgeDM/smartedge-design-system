# SMART/EDGE Website UI Kit

High-fidelity recreation of the SMART/EDGE marketing website — hero, nav, services grid, case-study feature, CTA band, footer. Open `index.html` to see the full one-page scroll with all components composed together.

## Components

- `Nav.jsx` — sticky top nav with blurred cream background, primary lime CTA
- `Hero.jsx` — Almost-Black hero with oversized Chakra Petch display + lime slash
- `ServicesGrid.jsx` — 4-up service card grid on cream, purple icons
- `CaseStudyFeature.jsx` — full-bleed case-study row with metric readouts
- `CtaBand.jsx` — lime band punctuation between light sections
- `Footer.jsx` — dark footer with wordmark, sitemap, fine print

## Notes

No codebase was provided — components are built from the stated brand rules + the two provided logos. If SMART/EDGE has a live site, share the URL and components will be pinned to it.
